# Skills

## 包含的 Skills

- [commit](skills/commit/SKILL.md) - 生成符合 Conventional Commits 规范的 git commit message

## 安装

### 方式 1: 使用 npx add-skill（推荐）

```bash
npx add-skill zkl2333/skills
```

### 方式 2: 手动安装

将 skills 目录添加到 Claude Code 配置中。
